# c 冒泡排序最优算法代码

``` c
#include <stdio.h>
#include <stdlib.h>

#define N 8
 
void bubble_sort(int a[], int n){
	int i=n,lastchange=1,j=0;
	int temp;
	while(i > 1){
		lastchange=1;
		for(j=0;j<i;j++){
			if(a[j]>a[j+1]){
				temp = a[j+1];
				a[j+1] = a[j];
				a[j] = temp;
				lastchange = j+1;
			}
		}
		i = lastchange;
	}
}

int main(void){
	int num[N] = {2,5,3,4,1,9,8,10},i=0;
	bubble_sort(num,N);
	for(;i<N;i++){
		printf("%d\t",num[i]);
	}
	printf("\n");
	return 0;
}

```

>注：冒泡排序：每循环一次，只把最大的给冒上去，所以要用双层循环，来把所以的都冒了
